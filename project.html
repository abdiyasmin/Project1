Turtle Game

The Turtle Game created a number-guessing game where they attempt to locate an invisible Turtle on a 500x500 grid. The game involves using loops and methods to collect user guesses and provide feedback until the correct coordinates are found.

Time Conversion

a Java program focused on time conversion using unconventional units. The project involves completing methods in two classes: TimeConversion and TimeConversionDriver. It will prompt the user to enter a number of seconds and then output the equivalent values in various units, including decaseconds, jiffies, New York minutes, nanocenturies, and scaramuccis.

Bookstore Management System

The Bookstore Management System helps a bookstore owner manage inventory and memberships. The system will handle three product types (books, CDs, and DVDs) and offer regular and premium memberships, tracking payments and total spending for all members.

Fast Food Kitchen Simulation

The FastFoodKitchen project focuses on improving a fast food ordering system by using a CSV file to create dynamic orders instead of relying on hardcoded ones. It includes updating the program to manage starting and ending orders correctly and adding error handling for user input and file operations.

Pet Store Management System

The Pet Store Management System project involves creating a system for managing pets and customer memberships using object-oriented programming. Participants will add functions for completing purchases, updating inventory, and managing adoption events, while designing a hierarchy of pet classes (dogs, cats, exotic pets).
